Краткое описание:

| Аспект  | Описание  |
| ------------ | ------------ |
| Тип системы | Развлекательная платформа prediction market (рынок предсказаний) |
| Валюта | Виртуальная (не имеет реальной ценности) |
| Аналог | Polymarket (упрощенная версия) |
| Цель бизнеса | Развлечение → удержание → монетизация через рекламу |
| Клиенты | Web, Mobile, Telegram Bot (3 точки входа) |
| SLA по данным | Пониженный (не real-time критично, как при реальных деньгах) |
| Аудитория | B2C, массовый пользователь |


Основные показатели:

| Метрика | Значение | Обоснование | 
| ------------ | ------------ |------------ |
| DAU| 10,000,000| чтобы много) |
| Пиковый RPS (Read)| ~15,000 - 20,000| Просмотры лент, рынков, профилей. |
| Пиковый RPS (Write)| ~300 - 500| Ставки, изменения балансов. |
| Активных рынков| ~3,000 - 5,000| Столько обычно висит на Polymarket в любой момент времени. |
| Ставок в день| ~2,000,000| Если хотя бы 10% DAU делают по 2 ставки в день. |
| Рост БД (Bets)| ~1 млрд строк в год| Требует партиционирования (partitioning) в PostgreSQL. |
