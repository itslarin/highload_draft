Краткое описание:

| Аспект  | Описание  |
| ------------ | ------------ |
| Тип системы | Развлекательная платформа prediction market (рынок предсказаний) |
| Валюта | Виртуальная (не имеет реальной ценности) |
| Аналог | Polymarket (упрощенная версия) |
| Цель бизнеса | Развлечение → удержание → монетизация через рекламу |
| Клиенты | Web, Mobile, Telegram Bot (3 точки входа) |
| SLA по данным | Пониженный (не real-time критично, как при реальных деньгах) |
| Аудитория | B2C, массовый пользователь |
| Механизмы монетизации | Показ рекламы, продажа обезличенной статистики |
